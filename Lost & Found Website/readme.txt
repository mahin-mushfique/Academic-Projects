This is the source code of  Lost & Found Bangladesh project.

Instruction to run  this project-

1. Download the zip file
2. Extract the file to the htdocs folder into the Xampp folder
3. Now, open the xampp control panel and start the xampp program
4. Go to Mysql admin through xmpp control panel.
5. Create a database named "l&fbd".
6. Import the sql file from the project folder.

Thus it will nbe run without any error.
